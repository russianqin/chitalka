# Читалка Pro v2.9 产品文档

> 俄语阅读 + AI 智能词汇积累工具
> 单文件 HTML，无须服务器，直接浏览器打开即可使用

---

## 一、最新变化（v2.9）

### 添加书签时自动去重生词本

每次点击「添加书签」时，自动扫描生词本，对同一 `lemma`（原型）的多个词条，仅保留**最后添加**的那一条（按 `savedAt` 判断），其余自动删除。

- 按 `lemma` 字段严格相等去重（字母和重音符 `·` 完全一致才视为同一原型）
- `lemma` 为空的词条不参与去重，全部保留
- 去重后自动刷新生词列表、全文高亮和统计数字
- 若有词条被清理，Toast 提示清理数量

> 适合录入过程中同一原型多次收录的场景，保持生词本整洁。

### 侧边栏公众号引导

在侧边栏底部静默展示一行小字，引导用户关注公众号「灵感小王」：

- 文案：「更多俄语资源，欢迎关注 公众号「灵感小王」↗」
- 字号 11.5px，颜色低对比度，不干扰正常阅读
- 点击「灵感小王」弹出二维码弹窗，扫码即可关注
- 二维码随软件分发，无需联网获取

> 低调引导，不自动弹出，不影响阅读体验。

## 二、软件功能

### 2.1 阅读器

**文本加载**

- 支持拖放 `.txt` / `.text` / `.md` 等纯文本文件到页面
- 也可点击上传按钮选择文件
- 支持超大文件（内容全部存入内存，无额外依赖）

**段落结构**

- 原始段落间距完整保留
- 俄语词以浅绿色背景高亮显示
- 单词点击可查词并收录

**阅读控制栏**（顶部工具栏）

| 控件 | 功能 |
|------|------|
| 字体切换 | 衬线体 / 无衬线体 |
| 字号滑块 | 14px ~ 28px，拖动实时预览 |
| 行距滑块 | 1.2 ~ 2.5，步进 0.1，默认 2.0 |
| 加粗按钮 | 一键切换正文加粗，再点取消 |
| 书签按钮 | 打开书签管理面板，支持添加、跳转、删除书签 |
| 字数统计 | 显示文档总词数（非生词数） |

> 字体、字号、加粗状态自动存入 `localStorage`，刷新页面保持。

---

### 2.2 AI 智能收录 + 翻译

**单词收录**

1. 阅读时点击任意俄语单词
2. 弹出浮动工具栏，显示当前单词
3. 点击 **🤖 AI 收录** 按钮
4. 弹出收录面板，DeepSeek AI 并行生成以下内容：

| AI 生成字段 | 说明 |
|-------------|------|
| 原型（词典形式） | 动词变位 / 名词变格后的原形 |
| 英文释义 | 单词原型对应的英文翻译（简洁，1~3 个词） |
| 中文释义 | 精确释义，含语境义 |
| 词根词缀分析 | 前缀 / 词根 / 后缀逐层拆解 |
| 记忆技巧 | 联想记忆、同源词提示 |
| 原文例句 | 从文档中提取的完整句子 |
| 语法标注 | 标注词性、体、时、数、格等 |

5. 用户可编辑所有字段，确认后保存

**设置项**

- DeepSeek API Key（存储在浏览器本地，不上传）
- 模型选择：`deepseek-chat`（推荐）或 `deepseek-coder`
- 获取地址：[platform.deepseek.com](https://platform.deepseek.com)

**多词翻译**

1. 选中一段俄语文本（≥2 个词）
2. 弹出浮动工具栏，显示 **🌐 翻译** 按钮
3. 点击后调用 AI 翻译，直接展示中文译文
4. 无需保存，专注当前阅读理解

---

### 2.3 生词本

**列表模式**

- 显示所有已收录单词
- 顶部搜索框：支持按单词、释义全文搜索
- 单词按收录时间倒序排列
- 点击任意卡片展开详情

**详情面板**

- 显示完整学习资料（释义、词根、记忆技巧、例句、语法）
- 可手动编辑或删除该词条

---

### 2.4 导入 / 导出

**导入**（设置页面 → 数据管理 → 导入词汇）

- 支持 `.csv` 和 `.tsv` 文件
- 自动识别分隔符（逗号 / 制表符）
- 智能匹配列名（识别"单词/слово/word"、"释义/trans"等常见列名）
- 导入前显示预览，确认后执行
- 自动跳过重复单词和空行

**导出**（点击「💾 备份」按钮弹出导出面板）

| 格式 | 说明 |
|------|------|
| TXT（文本） | 可选择日期范围，包含完整字段（收录日期、单词、原型、英文释义、释义、词根分析、记忆技巧、原文例句），同时复制到剪贴板 |
| JSON 备份 | 支持按日期筛选，包含所有词汇和设置，可完整恢复 |

---

### 2.5 书签功能

**添加书签**

- 加载文档后，点击阅读工具栏的「🔖 书签」按钮
- 点击「➕ 添加书签」记录当前阅读位置
- 自动记录：添加日期（YYYY-MM-DD）、精确到秒的时间戳（HH:mm:ss）、当前阅读进度百分比

**书签列表**

- 显示所有书签：文件名、添加日期、添加时间、阅读进度百分比
- 点击任意书签条目跳转至对应位置
- 点击删除按钮移除书签

**导出日期筛选**

- 点击「💾 备份」按钮弹出导出面板
- 可选择开始/结束日期，留空则导出全部
- 提供「今天」「本周」快捷按钮
- 默认按收录日期升序导出

---

### 2.6 数据存储

- 所有数据存储在浏览器 `localStorage` 中
- Key：`chitalka_vocab_v4`（词汇）、`chitalka_settings_v3`（设置）、`chitalka_reading_v1`（阅读进度）、`chitalka_bookmarks`（书签）
- 多次版本升级自动向前兼容，旧版本 Key 数据会迁移
- 关闭标签页即清除数据，可通过 JSON 备份导出保存

## 五、快捷操作汇总

| 操作 | 方法 |
|------|------|
| 加载文本 | 拖放文件到页面，或点击上传按钮 |

| 添加书签 | 阅读工具栏点击「🔖 书签」→ 点击「➕ 添加书签」 |
